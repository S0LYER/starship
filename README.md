## Features
* Displaying the work with git and cubernetis
* Dynamic displaying OS (MacOS/Linux/Windows)
* Displaying some directories
* displaying programming languages like python, rust, nodejs, go, C++, C#, C, Lua, etc
* RAM usage
* Battery
* Time
* Git status

> [!NOTE]
>  All of them with spetial color of tablet

## Installation
```
https://github.com/S0LYER/starship
cd starship
cp starship.toml ~/.config
```

> [!IMPORTANT]
>  You need to have nerd font in your system. You should enable this font in your terminal if you install this config on Mac. You shouldn't do it if you use it on Linux
